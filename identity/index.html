Industry Best Practices

Identity Security Best Practices

Microsoft Zero Trust Identity Pillars:

  • Verify explicitly: Always authenticate and authorize based on all available data points
  • Use least privilege: Limit user access with JIT/JEA and risk-based policies
  • Assume breach: Minimize blast radius and segment access

CISA & NIST Recommendations:

  • Phishing-resistant MFA for all users (FIDO2, Windows Hello)
  • Block legacy authentication protocols (IMAP, POP3, SMTP AUTH)
  • Implement Conditional Access with device compliance
  • Monitor and protect privileged accounts with PIM
