ESET Decommission

EDR Track Months 8-12

Phased removal of ESET Protect and ESET Inspect from all endpoints after confirming MDE protection is fully operational.

Removal Prerequisites

Do Not Proceed Until
  • ☑ MDE deployed to 100% of endpoints
  • ☑ MDE sensor healthy on all devices
  • ☑ Detection comparison shows MDE parity or better
  • ☑ ASR rules in Block mode (critical rules)
  • ☑ SOC fully trained on M365 Defender console
  • ☑ 30-day burn-in with no critical issues

ESET Uninstall Methods

Method 1: ESET Protect Console (Recommended)

ESET Protect Steps
1. ESET Protect → Computers → Select target group
2. Tasks → New Task → Software Uninstall
3. Select: ESET Endpoint Security / ESET Inspect
4. Options:
   ☑ Automatically reboot if required
   ☑ Remove ESET Management Agent after uninstall
5. Schedule: Immediate or maintenance window
6. Deploy to selected group

Method 2: SCCM/Intune Application Removal

PowerShell Script
# Silent uninstall ESET Endpoint Security
$ESETPath = "C:\Program Files\ESET\ESET Security\esetuninstall.exe"
if (Test-Path $ESETPath) {
    Start-Process -FilePath $ESETPath -ArgumentList "/SILENT /FORCE" -Wait
}

# Remove ESET Management Agent
$AgentPath = "C:\Program Files\ESET\RemoteAdministrator\Agent\ERAAgent.exe"
if (Test-Path $AgentPath) {
    Start-Process -FilePath "msiexec.exe" -ArgumentList "/x {ESET_AGENT_GUID} /qn" -Wait
}

# Cleanup registry
Remove-Item -Path "HKLM:\SOFTWARE\ESET" -Recurse -Force -ErrorAction SilentlyContinue

# Reboot
Restart-Computer -Force

Method 3: Manual Removal (Last Resort)

Steps
1. Download ESET Uninstaller from ESET website
2. Boot into Safe Mode
3. Run ESETUninstaller.exe
4. Select product to remove
5. Reboot into normal mode
6. Verify removal: Check Programs and Features

Post-Removal Validation

PowerShell - Verify Clean Removal
# Check no ESET services running
Get-Service | Where-Object {$_.Name -like "*eset*" -or $_.Name -like "*ekrn*"}

# Check no ESET processes
Get-Process | Where-Object {$_.Name -like "*eset*" -or $_.Name -like "*egui*"}

# Check MDE is primary AV
Get-MpComputerStatus | Select-Object AMServiceEnabled, AntispywareEnabled, AntivirusEnabled

# Verify MDE exclusions for ESET are removed (no longer needed)
Get-MpPreference | Select-Object -ExpandProperty ExclusionPath | Where-Object {$_ -like "*ESET*"}
