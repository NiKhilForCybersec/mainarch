Agent Conflicts

Resolving conflicts between ESET and MDE during the coexistence period, plus AMA agent issues.

ESET + MDE Conflicts

SymptomCauseSolution
High CPU usageScanning each other's filesAdd mutual exclusions (see Coexistence)
Both quarantine same fileRace conditionSet one to alert-only for malware, or accept first-wins
ESET blocking MDE updatesESET detecting Defender as threatAdd MDE paths to ESET exclusions immediately
Application crashesBoth hooking same processesAdd application to exclusions in both products

Verify Exclusions Applied

PowerShell
# Check MDE exclusions
Get-MpPreference | Select-Object -ExpandProperty ExclusionPath
Get-MpPreference | Select-Object -ExpandProperty ExclusionProcess

# Verify both services running
Get-Service -Name "ekrn","WinDefend","Sense" | Format-Table Name, Status

# Check for conflicts in event log
Get-WinEvent -LogName "Microsoft-Windows-Windows Defender/Operational" -MaxEvents 50 | 
    Where-Object { $_.Message -like "*conflict*" -or $_.Message -like "*exclusion*" }
